kast

Description

KCast is a Free Open Source WireCast clone

Features

Production

  • Live compositing with multiple layers
  • Live switching
  • Titles
  • Transitions (Fade, Cut)
  • Picture-In-Picture
  • Pan Tilt Zoom control

Distribution

  • Create playlists
  • h.264 x264 encoding
  • RTMP streaming

Inputs

  • Video

    • Screen capture (Local, Remote,OpenGL)
    • RTMP
    • Web streams
    • IP cameras
    • Cameras: USB, FireWire
    • SmartPhones

  • Audio

    • Microphones: USB, XLR, Unbalanced
    • Audio mixers
